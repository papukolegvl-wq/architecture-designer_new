# Architecture Designer

Визуальный редактор архитектуры программного обеспечения с поддержкой AI-ассистента.

## 🚀 Быстрый старт

### Windows

1. **Двойной клик** на файл `Launch Web App.bat`
   - Скрипт автоматически установит зависимости (если нужно) и запустит приложение
   - Браузер откроется автоматически на `http://localhost:5173`

### Linux / macOS

1. Откройте терминал в директории проекта
2. Выполните:
   ```bash
   chmod +x start.sh
   ./start.sh
   ```

### Универсальный способ (любая ОС)

```bash
npm install  # только при первом запуске
npm run dev
```

После запуска откройте браузер по адресу: `http://localhost:5500`

## 📦 Требования

- **Node.js** версии 16 или выше
- **npm** (устанавливается вместе с Node.js)

## 🎯 Возможности

- ✨ Визуальное проектирование архитектуры
- 🤖 AI-ассистент на базе Google Gemini
- 💾 Сохранение и загрузка проектов
- 📤 Экспорт в draw.io
- 🎨 Множество типов компонентов (базы данных, API, контейнеры и др.)
- 🔗 Гибкая система связей между компонентами
- 📊 Сравнение технологий

## 🛠️ Дополнительные команды

- **Сборка для продакшена:**
  ```bash
  npm run build
  ```

- **Просмотр собранной версии:**
  ```bash

  ```

- **Запуск Electron версии:**
  ```bash
  npm run electron:dev
  ```

## 📝 Примечания

- Приложение автоматически сохраняет состояние в localStorage браузера
- Все файлы сохраняются в формате JSON
- Для использования AI-ассистента требуется API ключ Google Gemini
