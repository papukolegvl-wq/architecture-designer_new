# 🖥️ Запуск приложения с рабочего стола

## ✅ Ярлык установлен!

Ярлык **Architecture Designer** был установлен на ваш рабочий стол.

## 🚀 Как использовать

### Способ 1: Двойной клик по ярлыку
1. Найдите на рабочем столе ярлык **Architecture Designer**
2. Дважды кликните по нему
3. Откроется терминал, и приложение запустится автоматически
4. Браузер откроется автоматически с приложением на `http://localhost:5500`

### Способ 2: Правый клик → Запустить
1. Правый клик по ярлыку
2. Выберите "Запустить" или "Execute"

## 📝 Что происходит при запуске

1. ✅ Проверка зависимостей (если нужно, они установятся автоматически)
2. ✅ Запуск dev-сервера на порту 5500
3. ✅ Автоматическое открытие браузера
4. ✅ Готово к работе!

## ⚠️ Важные заметки

- **Терминал останется открытым** - это нормально, так как сервер должен работать
- **Не закрывайте терминал** - это остановит сервер
- Если сервер уже запущен, скрипт просто откроет браузер
- Для остановки сервера закройте окно терминала или нажмите `Ctrl+C`

## 🔄 Переустановка ярлыка

Если нужно переустановить ярлык, выполните:

Если автоматическая установка не сработала, скопируйте ярлык вручную:

```bash
cp /home/DN101187POV/architecture-designer/Architecture-Designer.desktop ~/Desktop/
chmod +x ~/Desktop/Architecture-Designer.desktop
```
