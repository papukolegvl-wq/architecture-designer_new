# Инструкция по сборке приложения для браузера

Для того чтобы собрать проект для запуска в браузере, используйте один из следующих файлов в зависимости от вашей операционной системы:

### Windows
Запустите файл:
`build-browser.bat`

### Linux / macOS
Запустите файл:
`build-browser.sh`

---

После завершения работы скрипта, готовые файлы для размещения на хостинге или открытия в браузере будут находиться в папке:
`dist/`
